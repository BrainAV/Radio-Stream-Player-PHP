# Release Notes - v2.3.0 (The Strategic Baseline)

## 🎯 Overview
Version 2.3.0 marks a significant milestone in the project's history. It serves as the final, polished baseline for the LAMP-based (Linux, Apache, MySQL, PHP) open-source core, as the project shifts its focus toward a professional "Pro" architecture (v3.x) built on Node.js.

## ✨ What's New
- **Immersive Mode Layout Refresh** — Optimized the vertical alignment of the now-playing content in immersive mode. The station and track info are now perfectly centered in the space above the hero VU meters, providing a more balanced and high-end aesthetic.
- **Roadmap Strategic Realignment** — Performed a major cleanup of the project roadmap, moving future milestones (Wallpaper Search, Email Verification, Advanced Ads) to the upcoming v3.0 "Pro" project.

## 📦 Technical Changes
- **Service Worker (v2.3.0)** — Incremented cache versioning to ensure all users receive the new immersive layout adjustments.
- **Manifest Updates** — Formalized the version number for PWA installation.
- **Strategic Separation** — This repository now officially represents the "Open Source" core, with v3 development transitioning to a separate private/commercial ecosystem.

## 🚀 Future Vision (v3.0)
The upcoming "Pro" project will focus on:
- **Node.js Integration** for high-concurrency.
- **Stripe/PayPal** subscription models.
- **Dynamic Routing** for SEO-optimized station pages.
- **Cloud Background Uploads** and personal drive integration.
